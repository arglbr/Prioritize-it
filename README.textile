Bem vindo!

Esta planilha é uma ferramenta simples para guiar o trabalho de desenvolvimento de software.

Com ela é possível centralizar as necessidades de seus usuários e, com a ajuda deles, eleger quais precisam ser desenvolvidas primeiro.

Um pequeno guia de como usá-la pode ser lido no meu blog, onde relato minha primeira experiência usando-a:

h3. English

Welcome!

This spreadsheet is a simple tool to guide the work of software development.

With it you can centralize the needs of your users and, with their help, choose which ones need to be developed first.
